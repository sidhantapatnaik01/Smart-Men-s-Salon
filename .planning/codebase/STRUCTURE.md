# Codebase Structure
- `index.html`: Main HTML entry point. Includes CDN links and mounts the app.
- `app.jsx`: Contains all React components, form logic, and state management.
- `app.js`: Compiled output of `app.jsx` generated by esbuild.
- `images/`: Directory containing all static image assets (gallery, hero, services).
- `package.json`: Defines the build script (`build:local`) and tracks `esbuild` dependency.
